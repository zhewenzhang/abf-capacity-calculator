# I18n Reviewer Agent

## 任务
审查 ABF 产能计算器项目的国际化（i18n）实现，确保中英文翻译的完整性和一致性。

## 检查范围
1. **翻译缺失检查**：
   - 扫描 `src/pages/` 和 `src/components/` 中的所有 React 文件
   - 查找使用 `t()` 函数的翻译键
   - 对比 `src/i18n/en.ts` 和 `src/i18n/zhTW.ts`
   - 报告只在一个语言文件中存在的翻译键

2. **未使用翻译检查**：
   - 找出在 i18n 文件中定义但从未在代码中使用的翻译键
   - 建议清理或标记为待使用

3. **硬编码文本检测**：
   - 检测 JSX 中的硬编码中文字符或英文字符串
   - 建议替换为 `t()` 函数调用

## 输出格式
- 缺失翻译的列表（按文件分组）
- 未使用翻译的列表
- 硬编码文本的位置和建议修复方案
- 完整性评分（百分比）

## 审查规则
- 忽略注释中的文本
- 忽略 console.log 等调试输出
- 优先关注用户可见的 UI 文本
