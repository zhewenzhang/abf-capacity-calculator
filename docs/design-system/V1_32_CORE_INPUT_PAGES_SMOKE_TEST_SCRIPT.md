# v1.32 數據輸入頁面手動冒煙測試腳本

本文檔為 `v1.32.0` 的數據輸入頁面（產品、預測、產能）手動冒煙測試（Smoke Test）腳本，引導 QA 人員或開發者在重構完成後，在瀏覽器端進行直觀的視覺與功能校驗。

---

## 測試環境準備
1. 確保本地開發伺服器正常運行：`npm run dev`。
2. 啟動瀏覽器並打開系統。
3. 準備兩個測試帳號：
   - **帳號 A (Workspace Owner 或 Editor)**：具備完整寫入權限。
   - **帳號 B (Workspace Viewer)**：僅具備唯讀權限。

---

## 🧪 測試案例 1：產品管理 (`Products.tsx`) 價格格式化與 RWD 測試

### 測試步驟
1. 使用 **帳號 A** 登入系統。
2. 點擊左側導航欄「Products / 產品管理」。
3. 觀察產品列表中 SKU 的單價（Unit Price）顯示與對齊。
4. 將瀏覽器視窗寬度拉窄至 `375px`（手機視口模式）。

### 預期結果
- [ ] 產品列表中的 SKU 單價一律呈現為靠右對齊，且帶有千分位分隔（如 `$1,200.00 USD` 或 `35,000.00 TWD`）。
- [ ] 頁面頂部載入標準的 `<PageHeader>` 區塊，標題與描述清晰。
- [ ] **RWD 測試**：在 `375px` 窄屏下，產品表格外圍產生局部水平滾動條，欄位無擠壓重疊，整個頁面寬度未被撐大。

---

## 🧪 測試案例 2：Products Spreadsheet Lab 批量編輯與保存測試

### 測試步驟
1. 點擊產品管理頁面的「Spreadsheet Lab / 實驗室」。
2. 觀察頁面頂部 Banner、操作列與數據格點。
3. 修改其中一個 SKU 的價格，並在末尾新增一個全新行，填寫 SKU Code。
4. 觀察 ActionBar 狀態與保存。

### 預期結果
- [ ] 頁面頂部顯示共用 `<ExperimentalBanner>`，Toolbar 中原橘色 Experiment Tag 已被移去，視覺簡潔。
- [ ] 保存按鈕與放棄按鈕相鄰靠左放置，在未修改時均呈 `disabled` 灰色。
- [ ] 當修改資料後，保存按鈕變藍並動態顯示修改個數（如 `Save (2)`），點擊「Save」後資料儲存成功，按鈕自動還原為 `disabled`。
- [ ] 單元格表格外層包裹於 `.spreadsheet-wrapper` 內，窄屏下局部滾動流暢。

## 🧪 測試案例 3：需求預測 (`Forecasts.tsx`) 篩選與 Segmented 切換測試

### 測試步驟
1. 點擊左側導航欄「Forecasts / 需求預測」。
2. 觀察過濾與篩選 Toolbar。
3. 觀察視圖切換控件。
4. 將瀏覽器視窗拉窄至 `375px`。

### 預期結果
- [ ] 篩選面板套用 `.abf-toolbar` 類，背景為純白，圓角對齊良好。
- [ ] 視圖切換控件成功替換為現代感的 `<Segmented>` 組件，圓角與整體風格契合。
- [ ] **RWD 測試**：在 `375px` 窄屏下，篩選工具欄的按鈕群與 Select 自動 wrap 折行，無溢出視口。

## 🧪 測試案例 4：產能規劃 (`CapacityPlan.tsx`) Discard 容錯功能測試

### 測試步驟
1. 點擊左側導航欄「Capacity / 產能規劃」。
2. 切換至「Month」或「Capacity Lab / 實驗室」。
3. 在格點中修改某個月的 Core 產能數值。
4. 觀察頂部 ActionBar。
5. 點擊「Discard / 放棄變更」按鈕。

### 預期結果
- [ ] 頂部 ActionBar 正確引入，`Save` 與 `Discard` 按鈕靠左相鄰。
- [ ] 修改數值後，`Save` 按鈕與 `Discard` 按鈕同時變為啟用狀態。
- [ ] 點擊「Discard」按鈕後，彈出放棄成功提示，格點內的數值一鍵還原為修改前的值，`Save`/`Discard` 按鈕自動還原為 `disabled`。

---

## 🧪 測試案例 5：唯讀模式 (Viewer) 只讀 Alert 與格點安全測試

### 測試步驟
1. 登出 **帳號 A**。
2. 使用 **帳號 B (Workspace Viewer)** 登入系統。
3. 依次點擊「Products」、「Forecasts」、「Capacity」三大輸入頁面及其實驗室 Lab 頁面。
4. 觀察頁面頂部。
5. 嘗試在 Spreadsheet Lab 頁面中雙擊單元格或複製貼上資料。

### 預期結果
- [ ] 所有六個頁面的頂部，均高亮渲染標準的藍色唯讀警告：*「您在此工作區為 Viewer 角色。編輯功能已被禁用。」*，樣式、字體與外邊距完全一致。
- [ ] 所有頁面的 Save、Discard、Batch Set、Delete 按鈕全部呈 `disabled` 灰色。
- [ ] 雙擊 DataSheetGrid 中的任何單元格，**絕對不會**進入編輯模式；嘗試使用 Ctrl+V 貼上數據，格點數值**絕對不會**發生任何改變。
