### 3. 右侧用户菜单 User Menu 整合 (检查项 3)
* **审计状态**：`✅ PASS`
* **详细发现**：
  * 重写了 `TopNav` 右侧的用户信息结构。整合设计了下拉菜单 `userMenuItems` 并绑定在 `<Dropdown>` 菜单上。
  * 将以下 5 个原本平铺在顶栏、极其占用空间的元素**彻底移入**了右侧 User Menu 菜单：
    1. **Workspace**：使用 `<WorkspaceSwitcher />` 下拉组件。
    2. **Role**（owner/editor/viewer）：使用圆角胶囊 Badge 在菜单中显示，且带有清晰的权限级别颜色区分（`owner` 绿、`editor` 蓝、`viewer` 灰）。
    3. **UID**：在菜单中限制显示前 8 位（IBM Plex Mono 字体），点击该行即可触发 `navigator.clipboard.writeText` 复制完整 UID，并伴随 AntD 的打勾 Icon 反馈。
    4. **Email**：作为用户身份说明置于菜单项首行，不可选中。
    5. **Logout**：设计为底部的红色危险操作选项。
  * 激活点击入口优化为 `<Avatar>` 图标加上截取后的用户 email 别名，极具现代 SaaS 界面的干净利落质感。

### 4. 顶栏主导航分流降噪 (检查项 4, 6)
* **审计状态**：`✅ PASS`
* **详细发现**：
  * 以前 13 个入口全平铺的拥挤架构被重塑。

  * 当用户处于低频页时，「更多」按钮自动展现 `twk-nav-item-active` 激活样式，主导航逻辑无懈可击。

### 5. 语言与币别紧凑化 (检查项 5)
