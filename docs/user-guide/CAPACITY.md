# 產能規劃 (Capacity) 使用手冊

## 頁面用途

產能規劃頁面用於設定各工廠每月的 Core 與 BU 面板日產能，為產能利用率計算與缺口分析提供供給端資料。

## 使用者角色

| 角色 | 可執行操作 |
|------|-----------|
| Owner | 新增/刪除工廠、編輯產能、批次操作 |
| Editor | 新增/刪除工廠、編輯產能、批次操作 |
| Viewer | 僅能檢視，無法修改 |

## 執行期間

支援 2026 年 1 月至 2030 年 12 月，共 60 個月。

## 基本操作流程

### 1. 工廠管理

**新增工廠**
1. 點擊「+ Add Factory」按鈕
2. 輸入工廠名稱
3. 點擊確認

**刪除工廠**
1. 選擇工廠
2. 點擊「Remove Factory」按鈕
3. 確認刪除（該工廠所有月份產能資料一併刪除）

### 2. 編輯單一月份產能

1. 在表格中找到目標月份
2. 直接輸入 Core Panel/Day 或 BU Panel/Day 數值
3. 數值即時更新（需點擊 Save 儲存）

### 3. 批次設定產能

**依年份/季設定**
1. 選擇多個月份（Shift+Click 或 Ctrl+Click）
2. 點擊「Batch Set」按鈕
3. 輸入目標數值
4. 所有選取月份套用相同數值

**Fill Forward（→→）**
1. 設定某一月份的產能
2. 點擊該月份的「→→」按鈕
3. 所有後續月份自動套用相同數值

### 4. 切換檢視模式

| 模式 | 說明 |
|------|------|
| Month | 顯示每月詳細數值 |
| Quarter | 顯示季合計（3 個月加總） |
| Year | 顯示年合計（12 個月加總） |

### 5. 產能趨勢圖表

- 切換至「Charts」分頁
- 查看三種圖表：
  - Core Panel/Day 趨勢
  - BU Panel/Day 趨勢
  - Monthly Capacity 總覽

## 常見錯誤

| 錯誤訊息 | 原因 | 解決方式 |
|----------|------|----------|
| Factory name required | 工廠名稱空白 | 輸入有效名稱 |
| Invalid number | 輸入非數字 | 輸入有效數值 |
| No changes to save | 未修改任何資料 | 先修改資料再儲存 |

## 與其他頁面的資料聯動

| 頁面 | 聯動關係 |
|------|----------|
| Parameters | 工作天數設定影響月產能計算 |
| Dashboard | 產能資料用於計算利用率與缺口 |
| Results | 產能用於月度分析與瓶頸識別 |

## 計算公式

```
月產能 = 日產能 × 工作天數
Core 月產能 = Core Panel/Day × Working Days
BU 月產能 = BU Panel/Day × Working Days
```

## 權限注意事項

- Viewer 無法新增、編輯、刪除任何產能資料
- 刪除工廠會同時刪除該工廠所有月份資料
- 工作天數預設為 22 天/月（可在 Parameters 調整）

## 不該做什麼

- ❌ 不要輸入負數產能
- ❌ 不要在未確認情況下刪除工廠
- ❌ 不要忘記設定工作天數
- ❌ 不要誤用 Fill Forward 導致後續月份被覆蓋
