# 營業目標 BP (BP Targets) 使用手冊

## 頁面用途

營業目標 BP 頁面用於設定年度營業目標（Business Plan），作為 Dashboard 與 Results 頁面 BP 達成率分析的基準。

## 使用者角色

| 角色 | 可執行操作 |
|------|-----------|
| Owner | 編輯、儲存、放棄變更 |
| Editor | 編編輯、儲存、放棄變更 |
| Viewer | 僅能檢視，無法修改 |

## 資料結構

- **單位**：百萬新台幣（Million TWD）
- **年份範圍**：2026 - 2040（共 15 年）
- **顯示方式**：橫向年份排列，類似 Excel

## 基本操作流程

### 1. 輸入營業目標

1. 進入 `/bp-targets` 頁面
2. 在目標年份欄位輸入數值
   - 範例：輸入 `1200` = 12 億新台幣
3. 變更後會顯示橘色「dirty」高亮

### 2. 儲存變更

1. 點擊「Save」按鈕
2. 系統驗證數值（不可為負數、非數字）
3. 儲存成功後更新快照

### 3. 放棄變更

1. 點擊「Discard」按鈕
2. 所有未儲存的變更還原為上次儲存的值

### 4. 清空某年份目標

1. 將該年份欄位清空
2. 點擊 Save
3. 該年份會從資料庫中移除（不會存成 NaN 或 0）

## 數值語意

| 數值狀態 | 語意 |
|----------|------|
| `1200` | 該年度目標為 12 億 TWD |
| `0` | 該年度目標明確設為 0 |
| 空白/null | 該年度未設定目標（不等於 0） |

## 常見錯誤

| 錯誤訊息 | 原因 | 解決方式 |
|----------|------|----------|
| Negative value is not allowed | 輸入負數 | 輸入 ≥ 0 的數值 |
| Invalid numeric value | 輸入非數字 | 輸入有效數值 |
| No changes to save | 未修改任何資料 | 先修改資料再儲存 |

## 與其他頁面的資料聯動

| 頁面 | 聯動關係 |
|------|----------|
| Parameters | BP 目標已移至獨立頁面，Parameters 不再直接編輯 |
| Dashboard | BP 目標用於計算達成率、缺口 |
| Results | BP 目標用於 BP Analysis 與風險摘要 |

## 權限注意事項

- Viewer 無法編輯任何欄位
- Viewer 會看到「唯讀模式」警告
- Viewer 的 Save/Discard 按鈕為停用狀態

## 不該做什麼

- ❌ 不要輸入負數營業目標
- ❌ 不要輸入非數字字元
- ❌ 不要混淆「空白」與「0」的語意
- ❌ 不要嘗試從 Parameters 頁面編輯 BP（已移除）

## 重要提醒

1. **單位固定為百萬 TWD**：BP 目標以新台幣百萬元為單位，與 USD 營收比較時會自動換算
2. **Parameters 不覆蓋 BP**：在 Parameters 頁面執行 Save 或 Restore Defaults 不會影響 BP 目標
3. **空白 ≠ 0**：清空欄位會移除該年度目標，而非設為 0
