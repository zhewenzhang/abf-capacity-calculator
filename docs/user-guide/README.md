本目錄包含 ABF Capacity Calculator 各功能頁面的使用說明，協助使用者快速上手並正確操作系統。

| 文件 | 頁面 | 說明 |

## 適用角色

| 角色 | 權限 |
|------|------|
| Owner（擁有者） | 完整讀寫權限，可管理成員 |
| Editor（編輯者） | 可新增/編輯/刪除資料，無法管理成員 |
| Viewer（檢視者） | 唯讀，無法修改任何資料 |

## 快速開始

1. 使用 Google 帳號登入
2. 選擇「個人」或「共用工作區」
3. 依序設定：產品 → 預測 → 產能 → 營業目標
4. 查看 Dashboard 與 Results 分析結果

## 重要提醒

- 所有數字輸入欄位使用千分位顯示
- 營業目標單位為「百萬新台幣」
- 預測與產能資料支援 2026-2040 年
- Snapshot 一旦建立即無法修改（immutable）
