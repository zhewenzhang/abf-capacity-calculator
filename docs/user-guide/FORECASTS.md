# 預測管理 (Forecasts) 使用手冊

## 頁面用途

預測管理頁面用於輸入、編輯、批量產生各 SKU 的月度銷售預測數量（PCS），為產能分析提供需求資料。

## 使用者角色

| 角色 | 可執行操作 |
|------|-----------|
| Owner | 新增、編輯、刪除、批量產生、匯入匯出 |
| Editor | 新增、編輯、刪除、批量產生、匯入匯出 |
| Viewer | 僅能檢視，無法修改 |

## 執行期間

支援 2026 年 1 月至 2040 年 12 月，共 180 個月。

## 基本操作流程

### 1. 新增單筆預測

1. 點擊「+ New Forecast」按鈕
2. 選擇 SKU
3. 選擇月份（格式：YYYY-MM）
4. 輸入預測數量（PCS）
5. 系統自動帶入 SKU 的單價與幣別
6. 點擊「Save」儲存

### 2. 編輯預測

1. 在列表中找到目標預測
2. 點擊「Edit」按鈕
3. 修改數量或單價
4. 點擊「Save」

### 3. 刪除預測

1. 選取一或多筆預測
2. 點擊「Delete」按鈕
3. 確認刪除

### 4. 批量產生預測

**方式一：依年份快速產生**

1. 點擊「Generate」按鈕
2. 選擇目標年份
3. 選擇來源年份（複製該年資料）
4. 選擇目標 SKU（全部或指定）
5. 點擊「Generate」

**方式二：依成長率產生**

1. 點擊「Growth」按鈕
2. 選擇目標空年份
3. 輸入年成長率（%）
4. 系統會依據前一年同月資料乘以成長率產生
5. 已有資料的月份不會被覆蓋

### 5. 匯出預測

1. 點擊「Export」按鈕
2. 選擇匯出範圍（全部/特定年份）
3. 下載 Excel 檔案

## 常見錯誤

| 錯誤訊息 | 原因 | 解決方式 |
|----------|------|----------|
| SKU not found | 選擇的 SKU 不存在 | 先在 Products 頁面建立 SKU |
| Forecast PCS must be positive | 輸入負數 | 輸入 ≥ 0 的數值 |
| Month already exists | 該 SKU-月份已有預測 | 編輯現有資料或刪除後重建 |

## 與其他頁面的資料聯動

| 頁面 | 聯動關係 |
|------|----------|
| Products | 預測必須關聯已存在的 SKU |
| Dashboard | 預測總量顯示於 KPI，營收用於趨勢圖 |
| Capacity | 預測數量用於計算產能需求與利用率 |
| Results | 預測資料用於月度明細與 BP 達成率計算 |

## 權限注意事項

- Viewer 無法新增、編輯、刪除任何預測
- 批量產生會跳過已有資料的月份
- 刪除 SKU 不會自動刪除相關預測（但會影響計算）

## 不該做什麼

- ❌ 不要輸入負數預測數量
- ❌ 不要在未確認情況下批次覆蓋預測
- ❌ 不要建立無對應 SKU 的預測
- ❌ 不要忘記設定 SKU 的單價（影響營收計算）
